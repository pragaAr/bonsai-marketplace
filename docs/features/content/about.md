# Halaman About

## Tujuan

Memberikan informasi tentang marketplace bonsai, visi, dan tujuan platform.

## Fitur yang Tersedia

- Menjelaskan latar belakang platform.
- Menyampaikan nilai jual dari komunitas bonsai.
- Menjadi halaman statis yang mudah diakses dari navigasi utama.

## Catatan Implementasi

- Halaman ini dibuat sebagai komponen Livewire sederhana.
- Cocok untuk konten branding dan edukasi pengguna.
