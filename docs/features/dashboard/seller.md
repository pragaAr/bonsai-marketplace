# Dashboard Seller

## Tujuan

Memberikan panel operasional sederhana untuk penjual yang sudah disetujui.

## Fitur yang Diharapkan

- Daftar produk yang dimiliki seller.
- Status approval produk.
- Akses cepat ke fitur pengelolaan toko.

## Catatan Implementasi

- Halaman ini dirancang untuk mempermudah seller mengelola toko mereka secara terpusat.
