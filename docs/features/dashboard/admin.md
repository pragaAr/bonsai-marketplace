# Dashboard Admin

## Tujuan

Memberikan akses admin untuk memantau dan mengelola operasi marketplace.

## Fitur yang Diharapkan

- Ringkasan aktivitas sistem.
- Pengelolaan user dan akses.
- Peninjauan permintaan seller.
- Pengaturan role dan permission.
- Pemantauan produk yang masuk dan status approval.

## Catatan Implementasi

- Dashboard admin dikelola melalui Livewire.
- Akses dibatasi hanya untuk role `admin` dan `system_admin`.
