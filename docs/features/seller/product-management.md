# Manajemen Produk Seller

## Tujuan

Memungkinkan penjual mengelola produk yang dijual di marketplace.

## Fitur yang Diharapkan

- Menambahkan produk baru.
- Mengedit data produk.
- Mengunggah gambar produk.
- Melihat status persetujuan produk.
- Menghapus atau menonaktifkan produk.

## Catatan Implementasi

- Produk dikelola melalui model `Product`.
- Produk dapat memiliki status approval sebelum tampil di marketplace.
- Gambar produk disimpan ke media collection `images`.
- Admin dapat mengontrol proses approval produk.
