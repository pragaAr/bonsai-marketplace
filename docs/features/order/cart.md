# Keranjang Belanja

## Tujuan

Memungkinkan pengguna menampung produk yang ingin dibeli sebelum melakukan checkout.

## Fitur yang Tersedia

- Menambahkan produk ke keranjang.
- Melihat daftar produk yang dipilih.
- Mengubah kuantitas atau menghapus item.
- Menampilkan total harga sementara.

## Catatan Implementasi

- Fitur ini terkait erat dengan alur checkout.
- Keranjang biasanya dikelola secara session atau state aplikasi saat pengguna berinteraksi.
- Pengalaman pengguna dapat dikembangkan lebih lanjut dengan fitur penyimpanan keranjang antar sesi.
