# Koleksi Produk

## Tujuan

Menampilkan kumpulan produk dalam bentuk katalog yang mudah diakses pengguna.

## Fitur yang Tersedia

- Menampilkan daftar produk dalam grid atau list view.
- Mendukung filter dan pencarian.
- Menyediakan akses ke detail produk.

## Catatan Implementasi

- Halaman koleksi produk cocok dipakai untuk halaman shop.
- Produk dapat difilter berdasarkan kategori, status, dan fitur unggulan.
