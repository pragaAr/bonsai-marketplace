# Detail Produk

## Tujuan

Menampilkan informasi lengkap suatu produk bonsai dan mendukung keputusan pembelian pengguna.

## Fitur yang Tersedia

- Menampilkan nama, harga, deskripsi, dan kategori produk.
- Menampilkan gambar produk dari media library.
- Menyediakan informasi tambahan seperti spesies, care level, dan kebutuhan perawatan.
- Mendukung alur checkout dari halaman detail produk.

## Catatan Implementasi

- Halaman detail produk dikelola melalui Livewire.
- Gambar produk diambil dari koleksi media `images`.
- Jika media tidak tersedia, sistem menampilkan fallback image.
